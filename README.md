
# dagre

功能介绍


    1.基于插件 d3 及 dagre-d3的流程图展
    
    2.可进行节点的增删改功能

    3.可添加箭头描述

    4.可自定义箭头及节点背景色

    5.添加节点和动作tip
     
    6.添加流程展示（横向、竖向）---rankdir属性
